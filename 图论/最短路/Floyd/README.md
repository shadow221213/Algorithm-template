## 方法：

### 基本思路：
``` C++
for p in 枚举中转点
    for i in 枚举起点
        for j in 枚举终点
            g[i][j] = min(g[i][j], g[i][p] + g[p][j]);
```

### 复杂度
> 时间复杂度为$O(n^3)$

> 空间复杂度为$O(n^2)$

### 用途：
多源最短路、判负环
