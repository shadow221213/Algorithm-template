## 方法：

### 基本思路：
朴素`Dijkstra`算法基于贪心，通过维护一维的距离数组`dis`实现，`dist[i]`表示从源点出发到点`i`的最短距离。

朴素`Dijkstra`算法在每一次迭代中，都选择`dis`中值最小的点进行松弛操作，逐渐扩展最短路径范围。

### 复杂度
> 时间复杂度为$O(n^2)$，$n$为节点个数

> 空间复杂度为$O(n)$

### 用途：
单源最短路（稠密图更优）

## 例题：
*1334. 阈值距离内邻居最少的城市*
