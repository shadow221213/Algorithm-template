## 方法：

### 基本思路：
堆优化`Dijkstra`算法通过优先队列（堆）来优化朴素`Dijkstra`中的“找`dis`值最小的点”的过程。

### 复杂度
> 时间复杂度为$O(m\times log\ n)$

> 空间复杂度为$O(n)$

### 用途：
单源最短路（稀疏图更优）

## 例题：
*1334. 阈值距离内邻居最少的城市*
