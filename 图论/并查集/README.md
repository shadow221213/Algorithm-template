## 方法：

### 满足：
无向图

### 用途：
并查集用于判断连个点所在的集合是否属于同一个集合，若属于同一个集合但还未合并则将两个集合进行合并。同一个集合的意思是这两个点是连通的，直接相连或者通过其它点连通。

或

通过以下来判断环的个数：
``` C++
int res = 0;
for( int i = 0; i < m; i++ )
    if( i == uf.find(i) ) res++;
```
